Other water quality toolboxes

The links provide tools that can be used for computing different aspects related with the quality of water. These tools are developed independently by our partners and colleagues. The tools are:

  1. Surface water quality index and models (e.g., DO-Sag)
  2. Groundwater models (Prof. Al Valocchi, Univ-Illinois-UC)
  3. Aquifer-Storage-Recharge (ASR) models (Prof. Charles Werth, Univ-Texas, Austin)

These above toolboxes complement the toolboxes provided in this site.

User's guide & Documentation

The links below provide complete details required to use and develop the tools provided in this site. The suggested work-flow for using the tools is as follows:

  1. Start with the Database Toolbox
  2. Follow then to Analytical model Toolbox or Empirical model toolbox, and then
  3. Numerical model toolbox

The Decision Toolbox (not implemented yet) and the Other toolboxes do not require any work flow. The User's guide (not complete) will contain details on how to use the toolbox correctly with examples. Also, will include examples and case studies. The complete documentation will include also details on further development of the codes, adding new modules etc.
